**Статус:** ✅ Завершено (требуется применение SQL миграции)

**Что реализовано:**

#### 1. Расширение таблицы профилей
Добавлены поля для отслеживания прогресса пользователя:

#### 2. Новые таблицы базы данных

**`levels` - Определения уровней (10 уровней):**
- Уровень 1 "Новичок" (0 XP) 🌱
- Уровень 2 "Начинающий" (100 XP) 🌿
- Уровень 3 "Участник" (250 XP) 🌾
- Уровень 4 "Активист" (500 XP) 🌳
- Уровень 5 "Организатор" (1,000 XP) ⭐
- Уровень 6 "Эксперт" (2,000 XP) 💫
- Уровень 7 "Мастер" (4,000 XP) 🏆
- Уровень 8 "Гуру" (7,000 XP) 👑
- Уровень 9 "Легенда" (12,000 XP) 💎
- Уровень 10 "Титан" (20,000 XP) 🔥

**`achievements` - Определения достижений (17 достижений):**
- События: Первое событие, Организатор-10/50/100
- Социальные: Первое участие, Социальная бабочка-10/50/100, Первый отзыв, Критик
- Вехи: Уровень 5/10/25/50/100
- Специальные: Ранняя пташка, Популярный организатор, Идеальный рейтинг

**`user_achievements` - Прогресс по достижениям:**
- Отслеживание прогресса каждого пользователя
- Поддержка частичного прогресса
- Дата разблокировки

**`experience_log` - История получения опыта:**
- Все операции начисления/списания баллов
- Связь с событиями, отзывами, достижениями
- Причина начисления

#### 3. Система начисления баллов

**Автоматическое начисление:**
- **+50 XP** - Создание события
- **+20 XP** - Участие в событии (при подтверждении)
- **+10 XP** - Оставленный отзыв
- **+15 XP** - Получение положительного отзыва (4-5 звёзд)
- **+[rewards] XP** - Разблокировка достижения

**Триггеры:**
- `trigger_event_created_gamification` - автоматически начисляет XP при создании события
- `trigger_event_participation_gamification` - автоматически начисляет XP при участии
- `trigger_review_created_gamification` - автоматически начисляет XP за отзывы

#### 4. PostgreSQL функции

**`add_experience_points()`:**
- Начисляет опыт пользователю
- Автоматически пересчитывает уровень
- Записывает в лог операций
- Проверяет достижения при повышении уровня

**`check_and_unlock_achievement()`:**
- Разблокирует достижение для пользователя
- Начисляет награду за достижение
- Предотвращает повторную разблокировку

**`check_level_achievements()`:**
- Автоматически проверяет достижения, связанные с уровнями
- Вызывается при повышении уровня

```sql
INSERT INTO achievements (
  code, name_ru, name_en, description_ru, description_en,
  icon, color, category, rarity, points_reward, requirement
) VALUES (
  'marathon_organizer',

  'Create 5 events in one month',
  '🏃',
  '#f39c12',
  'events',
  'epic',
  200,
  '{"type": "events_in_period", "count": 5, "period_days": 30}'::jsonb
);
```

### Ручное начисление бонусных баллов

```sql
-- Администратор может начислить бонус
SELECT add_experience_points(
  'user_id',
  500,
  'admin_bonus',
  NULL,
  NULL,
  'Бонус за участие в бета-тестировании'
);
```

### Получение статистики пользователя

```sql
-- Полная информация о прогрессе пользователя
SELECT
  p.full_name,
  p.level,
  p.experience_points,
  l.name_ru as level_name,
  p.total_events_created,
  p.total_events_participated,
  p.total_reviews_given,
  COUNT(DISTINCT ua.id) as achievements_count
FROM profiles p
LEFT JOIN levels l ON p.level = l.level
LEFT JOIN user_achievements ua ON p.id = ua.user_id AND ua.is_unlocked = TRUE
WHERE p.id = 'user_id'
GROUP BY p.id, p.full_name, p.level, p.experience_points, l.name_ru,
         p.total_events_created, p.total_events_participated, p.total_reviews_given;
```
